== Description ==

This plugin will disable the functionalities of add new themes, plugins and file editing in your WordPress dashboard. If someone access your backend he cannot do anything to your website.

== Installation ==

1. Download and extract this plugin to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress, That's all!
